Это тестовый проект для создания датасетов из переписок в Telegram конкретного пользователя.

Сам экспортированный чат удалён и не включён в репозиторий.

Назначение: демонстрация преобразования Telegram export JSON в HTML и подготовки данных для датасетов.

Использование:
- Запустить `Html_to_read.py` для конвертации одного `messages.json` в `chat.html`.
- Запустить `tg_bulk_export_to_html.py` для пакетной обработки папки с экспортами.

Проект заброшен, ввиду нахождения более удобного инструмента
